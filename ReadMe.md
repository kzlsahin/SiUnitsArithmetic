# SI Units library with arithmetic capabilities.

There are basic types like milimeter or meter and combined types like square meter or square milimeter.

The project is open for contributions.

## Supported Units

Meter [m]

Milimeter [mm] 

Centimeter [cm]

SqMeter [m2] 

SqCentimeter[cm2] 

SqMilimeter[mm2]
